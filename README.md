## Kubernetes usefull command

Check minikube version :
```
Minikube version
```
Check kubernetes version :
```
kubectl version
```

Start minikube :
```
Minikube start
```

Get nodes :
```
kubectl get nodes
```

Get deployments :
```
kubectl get deployments
```

List resources :
```
kubectl get {resources}
```

```

```
